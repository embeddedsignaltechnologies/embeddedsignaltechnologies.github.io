Embedded Software Engineering Concepts

Real‑Time Operating System (RTOS) Concepts

Preemptive multitasking, task priorities, semaphores, mutexes, and message queues. Critical for deterministic behaviour in flight control & sensor fusion. ODST approach: minimize jitter, guarantee deadlines (rate-monotonic scheduling).

vTaskDelayUntil() • xSemaphoreTake() • event groups

Memory Management & DMA

Efficiently moving data between peripherals (ADC, I2S, SPI) without CPU overhead. Memory sections: .bss, .data, stack vs heap. Avoid fragmentation in constrained devices. DMA double-buffering for high‑throughput computer vision pipelines.

Low‑Power & Energy‑Aware Firmware

Sleep modes (stop, standby), wake-up sources, and DVFS. For CubeSat & drones, power budgeting is paramount. Use of low-power timers, event‑driven architecture, and efficient peripheral gating.

Hardware Abstraction Layer (HAL) & Drivers

Portable firmware using HAL (STM32Cube, Zephyr). Writing peripheral drivers: I2C, CAN, UART, PWM with callback-based IRQ management. Separation of business logic from hardware specifics.

Toolchain

GCC, CMake, OpenOCD, J-Link debugging, static analysis.

Testing strategy

HIL, SIL, unit testing with CppUTest, QEMU emulation.

CubeSat: Orbital Edge Computing

1U/3U architecture • On-board computer (STM32H7 / ARM Cortex-M7) • EPS with MPPT • Store-and-forward telemetry • Radiation-tolerant design patterns.

OBC + ADCS

Attitude determination using magnetometer, gyro, sun sensors. Reaction wheels / magnetorquers control loops implemented in embedded C.

Communications

UHF/VHF transceiver, AX.25 framing, CSP protocol, LoRa backup downlink. Frame error correction.

Flight Software

State machine (Safe, Detumble, Nominal, Standby), fault detection, FDIR with watchdog & memory scrubbing.

Quad‑Copter: Vision‑Aided Navigation

Computer Vision Stack: Onboard NVIDIA Jetson Nano / Raspberry Pi 4 with accelerated OpenCV, TensorFlow Lite. Real‑time object detection (YOLOv8), visual odometry, and precision landing.

Vision Pipeline

Image pre-processing, feature extraction (ORB/SIFT), Kalman filter for object tracking. Collision avoidance with depth estimation (monocular).

Flight Controller Fusion

PX4 / ArduPilot with companion computer MAVLink bridge. Sensor fusion: IMU + optical flow + GPS-denied navigation using CV.

Embedded CV Optimizations

NEON intrinsics, GPU shaders, model quantization (INT8), edge TPU acceleration for 4K real‑time analysis.
